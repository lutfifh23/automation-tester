# automation-tester
Automation tester using WebdriverIO

## Prerequisites

Node.js v20.12.2

npm v10.8.3

## Installation

 git clone

```bash
https://github.com/lutfifh23/automation-tester.git
```

initiation project

```bash
npm init -y
```

## Configuration

install [webdriverIO](https://webdriver.io/docs/gettingstarted/)

```bash
npm init wdio@latest .
```

## Running Test

To run test

```bash
npx wdio run ./wdio.conf.js
```
